COMO FAZER A MÚSICA TOCAR

1) Extraia o ZIP. Não rode o projeto de dentro do ZIP.
2) Coloque a música aqui:
   assets/musica-preferida.mp3

Atenção ao nome exato:
- musica-preferida.mp3
- tudo minúsculo
- com hífen
- sem acento
- sem espaço

Também aceito:
- assets/musica-preferida.m4a
- assets/musica-preferida.ogg

3) Abra pelo Live Server do VSCode ou por uma hospedagem HTTP.
   Evite abrir direto como file:/// no celular.

4) No celular, o navegador pode bloquear autoplay.
   Neste caso, toque no botão Tocar ou use o controle nativo do player.
